# dancestudioVANZ 予約サイト

LINEリッチメニューから飛ぶ予約サイトです。モバイルファーストで設計され、各クラスごとの曜日制限機能を備えたスムーズな予約体験を提供します。

## 🎯 機能

### レッスン選択
- **キッズダンス**: 4つのクラス（幼児・低学年・高学年・中学生）
  - 幼児クラス: 水曜・金曜 16:00-16:50
  - 低学年クラス: 金曜 17:00-18:00
  - 高学年クラス: 水曜・金曜 18:30-19:30
  - 中学生クラス: 水曜・金曜 19:40-20:40
- **HipHop**: 火曜・木曜 19:00-20:00
- **エアロビクス**: 土曜 10:00-11:00
- **癒しのヨガ**: 日曜 9:00-10:00

### 予約フロー
1. **レッスン選択**: 4つのカードから選択
2. **スムーズスクロール**: 選択したレッスンの予約セクションへ自動移動
3. **詳細入力**: レッスンに応じたフォーム入力
4. **LINE送信**: 予約内容をLINEトークに送信

### 技術仕様
- **ペライチデザイン**: 1ページで完結
- **レスポンシブ**: モバイル・タブレット・PC対応
- **スムーズスクロール**: 直感的なナビゲーション
- **曜日制限機能**: 各クラスごとの特定曜日のみ選択可能
- **フォームバリデーション**: 入力チェック機能
- **LINE連携**: LINEアプリ/Web版への自動送信

## 🔧 カスタマイズ

### 公式LINEアカウントの設定
`config.js`で公式アカウントIDを変更：
```javascript
LINE: {
    OFFICIAL_ACCOUNT_ID: '@あなたの公式アカウントID',
}
```

### レッスン時間の変更
`index.html`の各セクションで時間を編集：
```html
<p><strong>レッスン時間:</strong> 16:00-17:00</p>
```

### レッスン内容の追加
1. HTMLに新しいレッスンセクションを追加
2. CSSでスタイリング
3. JavaScriptでフォーム処理を追加

### LINE送信メッセージの変更
`script.js`の`formatBookingMessage`関数を編集：
```javascript
let message = '🎉 レッスン予約完了\n\n';
// メッセージ内容をカスタマイズ
```

## 🌟 主要機能

### スムーズスクロール
- レッスンカードクリックで該当セクションへ移動
- ヘッダー高さを考慮した正確な位置調整
- ハイライト効果で選択されたセクションを強調

### フォーム機能
- リアルタイムバリデーション
- 日付制限（今日から3ヶ月後まで）
- エラーメッセージ表示
- 入力内容の自動保存

### LINE連携
- モバイル: LINEアプリ直接起動
- PC: Web版LINEで開く
- 予約内容の自動整形
- 絵文字付きの見やすいメッセージ
