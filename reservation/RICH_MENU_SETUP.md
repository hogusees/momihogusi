# LINEリッチメニュー設定手順

## 📋 概要

LINE公式アカウントのリッチメニューに「予約ボタン」を設定して、予約フォームに連携させる手順です。

## 🎯 実装フロー

```
リッチメニュー「予約ボタン」タップ
↓
予約フォームURLに飛ぶ
↓
「予約をLINEに送信」ボタンタップ
↓
予約者のLINEトークに予約内容が送信される
```

## 🛠️ 設定手順

### **ステップ1: LINE Developers Consoleにアクセス**

1. [LINE Developers Console](https://developers.line.biz/console/) にアクセス
2. ログイン（LINE公式アカウントの管理者権限が必要）
3. 対象のプロバイダーを選択
4. 対象のチャネルを選択

### **ステップ2: リッチメニューを作成**

#### **2-1. リッチメニュー作成**
1. 左メニューの「Messaging API」をクリック
2. 「Rich menu」タブをクリック
3. 「Create」ボタンをクリック

#### **2-2. 基本設定**
- **Rich menu name**: `予約メニュー`
- **Chat bar text**: `予約はこちら`
- **Size**: `Full` (2500×1686px) または `Half` (2500×843px)

#### **2-3. エリア設定**
1. 「Add an area」をクリック
2. エリアをドラッグして配置
3. **Action type**: `Web page`
4. **URL**: 予約フォームの公開URL（例：`https://example.com/reservation/`）
5. **Open link in**: `In-app browser`

### **ステップ3: 画像の作成**

#### **3-1. 画像サイズ**
- **Full size**: 2500×1686px
- **Half size**: 2500×843px

#### **3-2. デザイン例**
```

│                                     │
```

#### **3-3. 画像作成ツール**
- **Canva**: リッチメニュー用テンプレートあり
- **Figma**: 無料デザインツール
- **Photoshop**: プロ用デザインツール

### **ステップ4: リッチメニューのアップロード**

1. 「Upload image」をクリック
2. 作成した画像ファイルを選択
3. 「Upload」をクリック

### **ステップ5: リッチメニューの公開**

1. 「Set as default」をクリック
2. 確認ダイアログで「OK」をクリック

## 🔧 詳細設定

### **エリア設定の詳細**

#### **予約ボタンエリア**
- **Action type**: `Web page`
- **URL**: 予約フォームのURL
- **Open link in**: `In-app browser`
- **エリアサイズ**: 十分な大きさ（タップしやすいサイズ）

#### **その他のボタン例**
- **お問合せ**: `Postback` → `action=contact`
- **アクセス**: `URI` → `https://maps.google.com/...`

### **複数エリア設定**

```javascript
// エリア設定例
{
  "areas": [
    {
      "bounds": {"x": 0, "y": 0, "width": 1250, "height": 843},
      "action": {
        "type": "web",
        "url": "https://example.com/reservation/"
      }
    },
    {
      "bounds": {"x": 1250, "y": 0, "width": 1250, "height": 843},
      "action": {
        "type": "postback",
        "data": "action=contact"
      }
    }
  ]
}
```

## 🧪 動作テスト

### **テスト手順**
1. 公式アカウントを友だち追加
2. リッチメニューが表示されることを確認
3. 「予約ボタン」をタップ
4. 予約フォームが開くことを確認
5. 予約フォームを入力
6. 「予約をLINEに送信」をタップ
7. LINEトークに予約内容が送信されることを確認

### **トラブルシューティング**

#### **リッチメニューが表示されない**
- 設定から公開状態を確認
- 友だち追加から24時間以内の場合、リッチメニューが表示されない場合あり

#### **予約フォームが開かない**
- URLが正しく設定されているか確認
- HTTPS対応しているか確認
- ブラウザのポップアップブロック設定を確認

#### **LINE送信が動作しない**
- スマホのLINEアプリがインストールされているか確認
- 予約フォームのJavaScriptが正常に動作しているか確認

## ⚠️ 注意事項

1. **URL設定**: 予約フォームのURLは正確に入力
2. **HTTPS必須**: セキュリティのためHTTPS対応必須
3. **画像サイズ**: 指定されたサイズで作成
4. **エリア配置**: タップしやすい位置に配置
5. **テスト必須**: 実際のスマホで動作確認

## 🎉 完成

設定完了後：
- リッチメニューから予約フォームへの連携
- 予約フォームからLINEトークへの送信
- 完全な予約フローが完成
